# Data Flywheel Map

## Flywheel Loops

### Loop 1: Correction loop: When your AI gets something wrong, what happens to that correction? Captured and reused → high. Discarded → low.
**Score: 2/5**


### Loop 2: Preference loop: Does the product learn individual/team preferences? Having analytics ≠ personalization.
**Score: 2/5**


### Loop 3: Domain context loop: Does usage in one area improve quality in adjacent areas? Siloed = 1, cross-domain transfer = 5.Model → Experience
**Score: 1/5**


### Loop 4: Network loop: Does each new user/team make the product better for everyone? Isolated = 1, strong network effects = 5.
**Score: 5/5**


**Total Flywheel Score: 10/20**
**Weakest Loop:**  Domain context loop: Does usage in one area improve quality in adjacent areas?
**Fix for weakest loop:** Break silo by reuse of common libraries, standards, reusable scripts, empower context loops and context mapping, asure common "language"/taxonomy is used.
